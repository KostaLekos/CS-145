##How to play:

####Step 1. Open a terminal or CMD for Windows and make sure that you have java 21 or greater installed.

####Step 2. Clone the repository: ```git clone https://github.com/KostaLekos/CS-145/```

####Step 3. Move into the folder: ```cd CS-145/```

####Step 4. Launch the game: ```java BlackJack.java```
